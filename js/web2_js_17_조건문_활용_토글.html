id="night_day":

HTML에서 id 속성은 요소의 고유 식별자를 지정합니다.
각 페이지 내에서 id 값은 유일해야 합니다.
id는 JavaScript나 CSS에서 특정 요소를 선택하고 조작하는 데 사용됩니다.


#night_day (JavaScript에서의 사용):

JavaScript에서 #은 id 선택자를 나타냅니다.
document.querySelector('#night_day')는 id가 'night_day'인 요소를 선택합니다.
#을 사용함으로써 JavaScript는 특정 id를 가진 요소를 빠르게 찾아 조작할 수 있습니다.


#의 기능:

HTML에서: id 속성 값으로 사용되어 요소에 고유 식별자를 부여합니다.
JavaScript에서: id 선택자로 사용되어 특정 id를 가진 요소를 선택합니다.
CSS에서도 #은 id 선택자로 사용되어 특정 id를 가진 요소의 스타일을 지정할 수 있습니다.
