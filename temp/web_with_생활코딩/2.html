선택자 = Selector → 태그 선택자, ID 선택자, class 선택자
선언, 효과 = Declaration → color:red;
속성 = Property → color
속성값 = Property Value → red

id ⊂ class ⊂ tag
id 선택자는 페이지 내에서 유일하게 한 번만 사용되어야 함!
class 선택자는 같은 교실을 사용하는 학생들의 모임! 같은 class를 적용시킬 태그에 class 선언!
tag 선택자에 style 효과를 적용할 경우 같은 태그를 쓰는 모든 영역에 효과 적용

id, class, tag 스타일이 동시에 적용될 때, 우선 순위는 id, class, tag 순으로 스타일 적용됨.
서로다른 색상 효과가 각각 들어있을 경우 id 선택자의 색상 효과만 적용됨.
a {
    color:black;
    text-decoration: none;
}
